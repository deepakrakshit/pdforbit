# PdfORBIT Frontend

This package contains the customer-facing Next.js application for PdfORBIT.

Use the root `README.md` as the primary project guide.

## Frontend Scope

- public landing pages
- tool pages
- pricing pages
- enterprise conversion paths
- secure billing API routes under `src/pages/api`

## Local Notes

- `.env.local`
- `.next/`
- `node_modules/`
- `tsconfig.tsbuildinfo`
