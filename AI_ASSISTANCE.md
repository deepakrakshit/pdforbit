# AI Assistance Disclosure

PdfORBIT is an AI-assisted project.

This disclosure is intentional.

AI tooling was used to accelerate parts of the software workflow, including:

- architecture exploration
- implementation drafting
- refactoring support
- test assistance
- documentation drafting
- deployment checklists
- repository cleanup planning

AI was not treated as the final authority.

Human review remained responsible for:

- product direction
- code acceptance
- deployment decisions
- production configuration
- payment credential handling
- final repository publishing

Project policy:

- no secrets should be committed
- generated code should be reviewed
- production claims should be verified
- behavior should be documented honestly
