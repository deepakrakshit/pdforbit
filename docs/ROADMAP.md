# Roadmap

## Near-Term

- Replace local filesystem storage with S3-compatible object storage for multi-instance artifact sharing.
- Add frontend component and end-to-end browser tests.
- Add richer plan-aware rate limits and billing enforcement.
- Introduce worker autoscaling and queue-depth monitoring.

## Planned Features

- S3-compatible object storage backend
- Autoscaling worker pool with queue-depth triggers
- Document AI features (smarter summarization, extraction, classification)
- Enterprise SSO integration
- Webhook-based job status callbacks for API consumers
- Usage dashboard with per-tool analytics
- Richer billing tiers and credit-based metering

## Long-Term

- Multi-region deployment support
- Public API with developer keys and rate limiting
- White-label support for enterprise customers
